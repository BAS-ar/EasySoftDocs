Asientos de apertura (en proceso de documentación)

El asiento de apertura de un ejercicio económico se genera en base al o a los asientos de cierre del ejercicio inmediato anterior.

Para acceder a la tarea tenés que ingresar al punto "Año fiscal", asegurarte que el ejercicio económico esté activo, abierto y luego editarlo:

Luego presioná  el botón...... y completá los datos que se muestran a la derecha.

Poner pantalla con el boton.....terminar.

Texto escrito por Haydee:

Para generar el asiento tenés que estar posicionado en el nuevo ejercicio, que tiene que estar activo y abierto, 
Posicionado en el nuevo ejercicio, el usuario deberá indicar el número del asiento de cierre en base al cual se generará, de manera automática, el asiento de apertura.
El sistema le ofrecerá una lista con todos los asientos de cierre del período anterior al actual (el cual deberá estar abierto), mostrará número de asiento y texto de observación, si lo tuviese.
Se crearán tantos asientos de apertura como fuere necesario, respondiendo cada uno a un asiento de cierre seleccionado.
